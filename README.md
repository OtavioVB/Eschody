# Eschody

O Eschody é uma plataforma open-source para estudantes que estão em processo de preparação para o vestibular e precisa de se manter organizado e produtivo até o dia da prova. A plataforma contará com mecanismos para conhecimento das principais dificuldades, conhecimento das lacunas de conhecimento e o que o estudante precisa para continuar se desenvolvendo.
